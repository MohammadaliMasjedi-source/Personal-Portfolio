# Contributing

This is the personal website & portfolio of **Mohammadali (Mo) Masjedi**. It's primarily a solo
project, but feedback and fixes are welcome.

## Ways to help
- **Spotted a bug or typo?** Open an issue describing what's wrong and where.
- **Have a suggestion?** Open an issue — design ideas, accessibility improvements, and performance
  tips are all appreciated.
- **Small fix?** Feel free to open a pull request.

## Running locally
No build step. It's a static, single-page site that deploys straight to GitHub Pages.

```bash
# from the repo root
python -m http.server 8080
# then open http://localhost:8080
```

## Conventions
- Keep it dependency-free and build-step-free — plain HTML/CSS/JS.
- Preserve the dark, cinematic, motion-rich aesthetic.
- Test on both desktop and mobile widths before submitting.
- Keep `index.html` deployable to GitHub Pages at all times (don't break `.nojekyll`).

## Code of conduct
Be respectful and constructive. That's it.
