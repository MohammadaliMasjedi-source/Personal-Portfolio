# Mo·MassJedi Portfolio — notes for Claude

(Style rules live in the global `~/.claude/CLAUDE.md`. This file is only **facts about this project**.)

## What it is
My personal brand site (*one mind, many lives*). Dark, cinematic, single-page.

## How it's built
- Plain **HTML + CSS + vanilla JS**. No Node, no bundler, no framework, **no build step**.
- Design system: `css/style.css`. Animations / interactions: `js/main.js`. Page: `index.html`.
- Deploys to **GitHub Pages** (GitHub Actions + `.nojekyll`).

## How to run it
- Just open `index.html` in a browser. (No build needed.)

## Good to know
- **Public-safe on purpose** — no private stuff (no employer, clients, finances, health).
- Respects `prefers-reduced-motion`.
- Public repo.
